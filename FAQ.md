# FAQ

## What is YouTube Auto Liker?

YouTube Auto Liker is a Chrome extension that helps automate liking YouTube videos based on the extension's settings.

## Where do I get support?

Open a support request in GitHub Issues:

https://github.com/kris-hamade/youtube-auto-liker-support/issues

## What should I include in a bug report?

Please include:

- What you expected to happen
- What actually happened
- Your Chrome version
- Extension version
- Whether the issue happens on all YouTube videos or only certain channels

Please do not include personal information, private YouTube account details, or sensitive browser data.

## Does this repository contain the extension source code?

No. This repository is only for support, documentation, issue tracking, and policy documents.
